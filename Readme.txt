The RGB LED matrix module is driven with serial data interface. It can be attached to any devices including Arduino via I2C and works as a I2C slave device, or via serial UART (can be attached to PC via a USB-TTL adapter). The module has 5¡Á8 character font built in as well as 3x5 and 3x7 digit font, so that numbers and letters can be displayed in color easily. The module is powered by 5V input and has a 6-pin connector providing SDA, SCL, GND, VCC, RxD, TxD breakout.
